# Dataset placement

Place the full e-commerce transaction dataset here and rename it:

```text
transactions.csv
```

Required columns:

- Transaction_ID
- Transaction_date
- Gender
- Age
- Marital_status
- State_names
- Segment
- Employees_status
- Payment_method
- Referral
- Amount_spent

The file `sample_transactions.csv` is only for a small test run. It is not enough for final portfolio evidence.
